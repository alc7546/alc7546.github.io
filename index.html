iOS App

Feb 2019 - Present

Music Geo Charts

Web Application

Dec 2018

I helped develop a web application that implemented external APIs to show the user different artist charts depending on the selected country.

I was responsible for hooking up the Google Maps API, implementing it with Vue.js to change the data displayed on the charts. I also implemented a basic database with Google Firebase to store the countries that users selected.

Software Used: HTML, JavaScript, CSS, Bootstrap, Vue.js, Visual Code, Git
